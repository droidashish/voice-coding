## To run the setup to do vibe coding using voice commands, do the following: 



1 First create a virtual environment using 

```
python3 -m venv venv
```

2 Activate the virual environment.

```
source venv/bin/activate
```

3 To install the related dependencies, do

```
pip install faster-whisper sounddevice pynput
```

4 To run the setup, run main.py file.

```
python main.py
```

Now, you with get video transcriptions wherever your cursor is, be it notepad or browser or some IDE.
